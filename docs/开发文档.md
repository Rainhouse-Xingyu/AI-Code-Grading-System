# AI Code Grading System 开发文档

## 一、项目概述

构建一个面向高校教学的AI自动评分系统，实现学生上传代码压缩包、教师发布作业与评分标准、AI自动评分、教师人工复核与修改、最终成绩发布给学生的完整流程。

### 核心能力

- 异步批量评分
- 多模型支持（DeepSeek + 本地 fallback）
- 评分标准结构化（Word/Excel 解析）

---

## 二、角色定义

### 学生端

- 提交作业
- 查看AI评分
- 查看教师最终评分
- 查看评分报告

### 教师端

- 创建作业
- 上传评分标准（Word/Excel）
- 批量发起AI评分
- 查看AI评分结果
- 修改评分
- 发布成绩

### AI系统（核心）

- 解压代码包
- 解析代码结构
- 调用 DeepSeek API
- 本地模型 fallback
- 生成评分报告

---

## 三、核心功能模块

### 1. 用户模块（User Module）

**功能：**

- 用户注册 / 登录
- 角色区分（student / teacher）
- JWT 认证

**能力：**

- 登录态管理
- 用户信息查询
- 班级信息管理

---

### 2. 作业管理模块（Assignment Module）

**教师功能：**

- 创建作业
- 发布作业
- 设置截止时间

**数据字段：**

- 作业标题
- 描述
- 发布教师
- 时间范围

---

### 3. 作业提交模块（Submission Module）

**学生功能：**

- 上传 zip 作业
- 查看提交记录

**后端能力：**

- 文件上传
- 文件存储（本地/MinIO）
- 解压准备

---

### 4. 评分标准模块（Rubric Module）

**教师功能：**

- 上传 Word 评分标准
- 上传 Excel 评分标准

**系统能力：**

- 解析 Word 文档
- 解析 Excel 表格
- 转换为 JSON 结构

**输出结构示例：**

```json
{
  "code_quality": 30,
  "functionality": 40,
  "innovation": 30
}
```

---

### 5. AI评分模块（核心模块）

**功能：**

- 调用 DeepSeek API
- 本地 fallback 模型
- Prompt 工程管理

**AI 输入：**

- 代码结构
- 评分标准 JSON

**AI 输出：**

- 总分（100分制）
- 分项评分
- 问题分析
- 改进建议
- Markdown 报告

---

### 6. 异步任务模块（Task Module）

**功能：**

- 提交AI评分任务
- Redis 队列管理
- 批量评分
- 重试机制

**状态流转：**

```
pending → running → success / failed
```

---

### 7. AI评分结果模块（Report Module）

**功能：**

- 存储AI评分结果
- 存储 Markdown 报告
- 存储结构化评分 JSON

**支持：**

- 多次AI评分记录
- 模型对比（DeepSeek vs fallback）

---

### 8. 教师批改模块（Review Module）

**功能：**

- 修改AI评分
- 修改评分说明
- 添加评语

**特点：**

- 保留AI原始结果
- 支持版本记录

---

### 9. 成绩发布模块（Publish Module）

**功能：**

- 发布最终成绩
- 学生可见
- 锁定评分结果

---

### 10. 文件管理模块（File Module）

**功能：**

- 上传 zip
- 上传 Word / Excel
- 文件存储管理

---

## 四、AI 服务模块（Python Service）

**功能：**

- 代码解压
- 代码分析
- 调用 DeepSeek API
- Prompt 生成
- Token 统计

---

## 五、系统核心流程

### 学生提交流程

```
上传zip → 保存文件 → 创建 submission
```

### 教师评分流程

```
创建作业 → 上传 rubric → 学生提交 → 创建 AI task
→ Redis 队列 → Python AI 服务 → DeepSeek 评分 → 生成 report
```

### AI 评分流程

```
解压代码 → 分析结构 → 加载 rubric JSON → Prompt 生成
→ 调用 DeepSeek → 返回评分结果 → 存数据库
```

---

## 六、数据库核心实体

| 实体 | 说明 |
|------|------|
| User | 用户（学生/教师） |
| Assignment | 作业 |
| Submission | 提交记录 |
| Rubric | 评分标准 |
| AiTask | AI评分任务 |
| AiReport | AI评分报告 |
| TeacherReview | 教师批改记录 |
| GradePublish | 成绩发布 |
| File | 文件管理 |

## 七、技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Spring Boot 3 + JDK 17 + MyBatis Plus |
| 缓存/队列 | Redis |
| 数据库 | MySQL |
| AI 服务 | Python FastAPI + DeepSeek API |
| 前端 | Vue3 + Vite |
| 存储 | 本地 / MinIO |
